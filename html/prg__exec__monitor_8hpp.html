Description détaillée

Entry point for the end user into the Program Execution Monitor.

Use this header to forward declare function prg_exec_monitor_main and to automatically define a main function for you. If you prefer to use your own main you are free to do so, but you need to define BOOST_TEST_NO_MAIN before incuding this header. To initiate your main program body execution you would use statement like this:

::boost::prg_exec_monitor_main( &my_main, argc, argv );

Also this header facilitate auto linking with the Program Execution Monitor library if this feature is supported
