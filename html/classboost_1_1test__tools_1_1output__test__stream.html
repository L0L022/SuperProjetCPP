Constructor

Paramètres
[in]pattern_file_nameindicates the name of the file for matching. If the string is empty, the standard input or output streams are used instead (depending on match_or_save)
[in]match_or_saveif true, the pattern file will be read, otherwise it will be written
[in]text_or_binaryif false, opens the stream in binary mode. Otherwise the stream is opened with default flags and the carriage returns are ignored.
